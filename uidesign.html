UI Design

User interface design

Sketches

We initially designed several wireframe sketches that would smoothen the transition to a higher-fidelity interactive prototype.

We considered the possible idea of a timeline or a checklist system, so it was easy to track each stage of the application process and so the users are not overwhelmed by a large form of information. We iterated this process of multiple sketches in order to obtain various designs.

Submission Portal Sketches

The sketches above tried to encompass a user-friendly interaction which make the user experience easy to follow and a fluid process. We felt the submission portal was an integral part of the user experience.

Feedback System Sketches

We designed multiple sketches for the reviewing system considering a red-amber-green system for feedback as well as a point-based system. Users expressed they liked the red-amber-green method with a combination of point based as it allowed easy comparison of projects.

Prototype
